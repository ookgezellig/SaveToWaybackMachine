# Saving KB-managed websites to the Wayback Machine

# Updated wbm-archiver script - 15-7-2021
On 15-7-2021 I have checked if the [wbm-archiver script from 2 years ago](Literatuurplein/scripts/wbm-archiver/SaveLiteratuurpleinToWaybackMachine.py) was still working. It turned out this was not the case, so I did a small [update of the script]() to make it work again. However, this script is still not 100% stable or trustworthy and ideally should be checked by a proper developer with knowlegdge about the WBM and the [Memento protocol](http://mementoweb.org/guide/rfc/) that is used in the WBM. 

# Goals
Some websites managed by the KB will change or be discontinued in the near future.

To preserve the current content of the site (e.g. for sourcing Wikipedia articles) a copy was submitted to The Wayback Machine (web.archive.org).
