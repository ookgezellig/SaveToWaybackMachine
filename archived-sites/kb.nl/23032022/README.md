# kb.nl Archive - March 2022 (New Site)

*Archived: 23-24 March 2022*

## About

This folder contains URLs archived from the **new kb.nl website** shortly after its relaunch on 17 March 2022.

On the same day, a new dedicated [KB special collections website](https://collecties.kb.nl) also went live. Both sites were archived.

## Contents

| File | Description |
|------|-------------|
| [url-inputlist_23032022.txt](url-inputlist_23032022.txt) | Input list of URLs to be archived |
| [urls_kbnl_archivedwbm_23032022.csv](urls_kbnl_archivedwbm_23032022.csv) | Archived URLs with Wayback Machine links |
| [urls_kbnl_archivedwbm_23032022.xlsx](urls_kbnl_archivedwbm_23032022.xlsx) | Excel version of the archived URLs |

The CSV/XLSX files contain Wayback Machine URLs in the format:
```
https://web.archive.org/web/[TIMESTAMP]/[ORIGINAL_URL]
```
